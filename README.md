# NodeWorks Legal Site

This repository hosts the GitHub Pages website for the NodeWorks Discord bot legal documents.

## Pages Included

- [Home](./)
- [Privacy Policy](./privacy-policy)
- [Terms of Service](./terms-of-service)

## Enable GitHub Pages

1. Open your repository on GitHub.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Choose branch **main** and folder **/(root)**.
5. Save changes and wait for deployment.

## Edit Placeholders

Before publishing, update the following placeholders in both legal pages:

- `[CONTACT_EMAIL]` → your legal/privacy contact email
- `[WEBSITE_URL]` → your public website URL (or leave blank)
- `[EFFECTIVE_DATE]` → the date your terms/policy take effect
